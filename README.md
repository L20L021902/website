## Authentication

- Method: `POST`
- Endpoint: `/login`
- Body:
```
{
	"username": "admin",
	"password": "password"
}
```

## Add goods
- Method: `POST`
- Endpoint: `/employee/add`
- Body:
```
{
    "goods_id": 567001231,
    "name": "Apple",
    "category": "电子产品",
    "buy_price": 5999,
    "sell_price": 6399
}
```
